Que es Nim ?

Nim es un nuevo lenguaje de programacion multi-proposito multi-paradigma compilado de tipado fuerte estatico con inferencia, genera binarios nativos ejecutables autocontenidos sin dependencias de 1 solo archivo y ademas puede correr en el navegador web, tiene sintaxis similar a Python, posee metaprogramacion muy poderosa, puede importar librerias C y JavaScript, lleva menos lineas de codigo que lenguajes de alto nivel, sintaxis muy expresiva, manejo de memoria multi-paradigma incluyendo 8 Garbage Collectors, Life-Tracking ARC Deterministico y manejo de memoria manual, compila a C puro optimizado para Backend, compila a JavaScript para Frontend, y puede correr interpretado para DevOps. Puede usarse en cualquier cosa que pueda compilar C, desde microcontroladores Arduino a servers de produccion.

Que es esta web ?

Es un servicio de Playground y Pastebin de codigo, para practicar, aprender y compartir codigo, tiene 120+ Ejemplos, puede usarse desde movil o PC. El servidor es 0.5 CPU y 256 Megabytes de RAM con Linux, por consiguiente no todas las opciones de Nim estan habilitadas, si eres una empresa y quieres Sponsorearnos ponte en contacto. Todo el codigo es Libre. Todo el servicio es Gratis. Links expiran luego de 99 dias.
